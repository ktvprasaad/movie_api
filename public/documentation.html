WebFlix - Movies API

  • URL
  • Info
  • Parameters

The web application provides the users to access the information about different movies, directors, and genres. Users may sign up, update their personal information, create and maintain their favorite movies list. Besides that it provides the users about the main url, authentication and parameters using that how to perform a lot of operations with the WebFlix Online Movie App.

Resource URL

WebFlix Movie Api

Resource Information

Resource formats JSON
Requires Authentication? Yes
Rate Limited? No
