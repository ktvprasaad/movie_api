It's a web application that provides the users to access the information about different movies, directors, and genres. Users may sign up, update their personal information, create and maintain their favorite movies list. Besides that it provides the users about the main url, authentication and parameters using that how to perform a few other operations with the WebFlix Online Movie App.

Register New User

HTTP Method

POST

Path/URL

/users

Request Body Data Format

Example of a JSON object holding data
about the users to be added, structured
like :

{
    "username": "Leo",
    "password": "abc123XYZ",
    "emailID": "leo_tall@gmail.com",
    "birth": new Date("2001-01-15")
}

Response Body Data Format

Example of a string holding data
about the users that was added :

Congratulations Leo! Your account has been created.

    "Email": "chocochip@yahoo.co.in",
    "Birthday": "2009-12-11T00:00:00.000Z",

        "5dba417df90a580f64bd30ea"
    ]
}

Update User's Personal Details

HTTP Method

PUT

Path/URL

Request Body Data Format

Example of a JSON object holding data about
the users to be updated, structured like :

{
    "password": "asdfa1310!",
    "emailID": "vjxyz@gmail.com",
    "birth": "1983-01-15T00:00:00.000Z"
}

Below message will be sent as response :
Your personal details have been updated.
